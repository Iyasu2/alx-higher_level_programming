``print_square`` prints a square with the character #, using 
the argument size

::

    >>> import importlib

    >>> module = importlib.import_module('4-print_square')
    >>> print_square = module.print_square
    >>> print_square(4)
    ####
    ####
    ####
    ####

size must be an integer

::

    >>> print_square("strwf")

::

    >>> print_square(-20)

with the message size must be an integer

::

    >>> print_square(-10.2)
